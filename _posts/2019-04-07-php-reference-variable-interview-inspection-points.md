在php的基础里，引用变量是一个重要的只是点。这里将详细介绍引用变量。

<!-- more -->

# 引用变量的概念

使用不同的名字访问同一个变量内存


# 引用变量的定义方式

在变量前加 & 符号

# 工作原理
COW 机制 copy on write

## 先不使用引用变量
```shell
 $a = range(0,5); 
 $b = $a;
```
将a变量值赋给b变量，这样触发COW机制，a b两个变量指向一个内存空间。

```shell
$b = range(1,6);
``` 
 当改变b变量的值时，这样触发COW机制，会生成 b 的内存空间，a b两个变量拥有各自的内存空间。

## 使用引用变量

```shell
$b = &$a; 
$b = range(1,6);
```
这样不会触发COW机制，引用赋值两个变量共享一个内存空间，修改的共享空间内容。
